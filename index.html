2 Resumen de Intervalos de Confianza

Los intervalos de confianza permiten estimar un parámetro poblacional mediante un rango de valores con cierto nivel de confianza.

Para distribuciones simétricas:

\hat{\theta} \pm MoE

donde:

  • (\hat{\theta}): estimador muestral
  • (MoE): margen de error
  • (SE): error estándar
  • (1-\alpha): nivel de confianza

El margen de error se calcula como:

MoE = SE \cdot \text{valor crítico}

3 1. Media poblacional con varianza conocida

Se utiliza la distribución normal (Z).

\bar{x} \pm z_{\alpha/2}\frac{\sigma}{\sqrt{n}}

Supuestos

  • Varianza poblacional conocida.
  • Muestra aleatoria.
  • Población normal o muestra grande.

Ejemplo

Peso promedio de dispositivos electrónicos.

  • (n=50)
  • (\bar{x}=75)
  • (\sigma^2=25)
  • Nivel de confianza: 95%

Intervalo:

[73.6141,\ 76.3859]

Interpretación

Con 95% de confianza, la media poblacional del peso está entre 73.61 y 76.39 gramos.

4 2. Media poblacional con varianza desconocida

Se utiliza la distribución t de Student.

\bar{x} \pm t_{(\alpha/2,n-1)}\frac{s}{\sqrt{n}}

Supuestos

  • Varianza poblacional desconocida.
  • Datos aproximadamente normales.
  • Muestra aleatoria.

Ejemplo

Pesos de 10 dispositivos electrónicos.

Intervalo:

72.61414,\ 77.78586

Interpretación

Con 95% de confianza, la media verdadera está entre 72.61 y 77.79 gramos.

5 3. Proporción poblacional

\hat{p} \pm z_{\alpha/2}\sqrt{\frac{\hat{p}(1-\hat{p})}{n}}

Supuestos

  • Muestra aleatoria.
  • (n\hat{p} \ge 5)
  • (n(1-\hat{p}) \ge 5)

Ejemplo

Encuesta de preferencia de producto.

  • (n=200)
  • 150 prefieren el producto.

Intervalo al 99%:

0.671,\ 0.829

Interpretación

La proporción verdadera de personas que prefieren el producto está entre 67.1% y 82.9%.

6 4. Varianza poblacional

Basado en la distribución chi-cuadrado.

Límite inferior:

L_i=\frac{(n-1)s^2}{\chi^2_{(\alpha/2,n-1)}}

Límite superior:

L_s=\frac{(n-1)s^2}{\chi^2_{(1-\alpha/2,n-1)}}

Supuestos

  • Población normal.
  • Muestra aleatoria.

Ejemplo

Peso de bolígrafos.

Varianza muestral:

s^2=0.0005771264

Intervalo al 95%:

0.0003660505,\ 0.001042973

Interpretación

La varianza verdadera del peso de los bolígrafos está entre 0.000366 y 0.001043.

7 5. Diferencia de medias pareadas

Se trabaja con diferencias:

d_i = X_{antes} - X_{después}

Intervalo:

\bar{d} \pm t_{(\alpha/2,n-1)}\frac{s_d}{\sqrt{n}}

Supuestos

  • Diferencias aproximadamente normales.
  • Datos pareados.
  • Muestra aleatoria.

Ejemplo

Presión arterial antes y después de tratamiento.

Resultados:

  • Media diferencias: (7.6)
  • Desviación estándar: (1.264911)

Intervalo:

6.695137,\ 8.504863

Interpretación

El tratamiento reduce la presión arterial entre 6.70 y 8.50 mmHg aproximadamente.

Como el intervalo no contiene el 0, existe diferencia significativa.

8 6. Diferencia de medias independientes

Caso A: Varianzas conocidas

(\bar{X}_1-\bar{X}*2) \pm Z*{\alpha/2}\sqrt{\frac{\sigma_1^2}{n_1}+\frac{\sigma_2^2}{n_2}}

Supuestos

  • Muestras independientes.
  • Varianzas conocidas.
  • Poblaciones normales o muestras grandes.

Ejemplo

Curso virtual vs presencial.

Intervalo:

1.421546,\ 8.578454

Interpretación

Los estudiantes del curso virtual obtienen entre 1.42 y 8.58 puntos más.

Caso B: Varianzas desconocidas pero iguales

Primero se verifica si las varianzas pueden considerarse iguales mediante el intervalo para:

\frac{\sigma_1^2}{\sigma_2^2}

Si el intervalo contiene 1, se consideran iguales.

Ejemplo

Intervalo para cociente de varianzas:

0.2703474,\ 4.137484

Como contiene 1:

  • Varianzas iguales.

Intervalo para diferencia de medias:

1.263404,\ 10.00326

Interpretación

Existe diferencia significativa entre ambos grupos.

Caso C: Varianzas desconocidas y diferentes

Cuando el intervalo del cociente de varianzas NO contiene 1.

Se utiliza la aproximación de Welch.

Supuestos

  • Muestras independientes.
  • Normalidad aproximada.
  • Varianzas distintas.

9 7. Diferencia de dos proporciones

(\hat{p}_1-\hat{p}*2) \pm z*{\alpha/2}\sqrt{\frac{\hat{p}_1(1-\hat{p}_1)}{n_1}+\frac{\hat{p}_2(1-\hat{p}_2)}{n_2}}

Supuestos

  • Muestras independientes.
  • Tamaños suficientemente grandes.
  • Muestras aleatorias.

Ejemplo

Clientes satisfechos en dos tiendas.

Intervalo:

-0.0822251,\ 0.0822251

Interpretación

El intervalo contiene 0, por lo tanto no existe evidencia suficiente para afirmar diferencias significativas entre las proporciones de satisfacción de ambas tiendas.

10 8. Intervalo de confianza para la proporción de la población

Se utiliza cuando queremos estimar el porcentaje o proporción de individuos que poseen cierta característica.

Fórmula

\hat{p} \pm z_{\alpha/2}\sqrt{\frac{\hat{p}(1-\hat{p})}{n}}

donde:

  • (\hat{p}): proporción muestral
  • (n): tamaño de muestra
  • (z_{\alpha/2}): valor crítico de la normal estándar

Ejemplo

En una encuesta a 400 personas, 260 afirmaron preferir una marca de café.

  • (n=400)
  • (x=260)
  • (\hat{p}=260/400=0.65)

Nivel de confianza: 95%.

Resultado

IC=0.6033,\ 0.6967

Interpretación

Con 95% de confianza, la verdadera proporción de personas que prefieren la marca de café está entre 60.33% y 69.67%.

11 9. Intervalo de confianza para la varianza de la población

Se utiliza para estimar la variabilidad poblacional.

Fórmulas

Límite inferior:

L_i=\frac{(n-1)s^2}{\chi^2_{(\alpha/2,n-1)}}

Límite superior:

L_s=\frac{(n-1)s^2}{\chi^2_{(1-\alpha/2,n-1)}}

donde:

  • (s^2): varianza muestral
  • (n): tamaño de muestra

Ejemplo

Se toma una muestra de 20 baterías y se obtiene:

  • (s^2=16)
  • (n=20)

Nivel de confianza: 95%.

Resultado

IC=9.14,\ 34.58

Interpretación

Con 95% de confianza, la verdadera varianza poblacional está entre 9.14 y 34.58.

12 10. Intervalo de confianza para el cociente de las varianzas de dos poblaciones

Permite comparar la dispersión entre dos poblaciones.

Fórmula

\frac{s_1^2/s_2^2}{F_{(1-\alpha/2,n_1-1,n_2-1)}} < \frac{\sigma_1^2}{\sigma_2^2} < \frac{s_1^2/s_2^2}{F_{(\alpha/2,n_1-1,n_2-1)}}

donde:

  • (s_1^2,s_2^2): varianzas muestrales
  • (F): distribución F de Fisher

Ejemplo

Dos máquinas producen piezas metálicas.

  • Máquina A: (n_1=15,\ s_1^2=25)
  • Máquina B: (n_2=12,\ s_2^2=10)

Nivel de confianza: 95%.

Resultado

IC=0.76,\ 8.91

Interpretación

Como el intervalo contiene al número 1, no hay evidencia suficiente para afirmar que las varianzas poblacionales sean diferentes.

13 11. Intervalo de confianza para la diferencia de proporciones de dos poblaciones

Se utiliza para comparar porcentajes entre dos grupos.

Fórmula

(\hat{p}_1-\hat{p}*2) \pm z*{\alpha/2}\sqrt{\frac{\hat{p}_1(1-\hat{p}_1)}{n_1}+\frac{\hat{p}_2(1-\hat{p}_2)}{n_2}}

Ejemplo

Se comparan dos métodos de enseñanza.

  • Grupo A: 90 de 150 aprobaron.
  • Grupo B: 70 de 140 aprobaron.

Entonces:

\hat{p}_1=0.60

\hat{p}_2=0.50

Nivel de confianza: 95%.

Resultado

IC=[-0.010,\ 0.210]

Interpretación

Como el intervalo contiene el 0, no existe evidencia suficiente para afirmar una diferencia significativa entre las proporciones de aprobación de ambos métodos.

14 .Ejercicio 1 - Media de la población (varianza poblacional conocida)

Un fabricante produce anillos para los pistones de un motor de automóvil. Se sabe que el diámetro del anillo esta distribuido aproximadamente de manera normal, y que tiene una desviación estándar = 0.001 mm. Una muestra aleatoria de 15 anillos tiene un diámetro promedio de x = 74.036 mm. Construya un intervalo de confianza bilateral del 99% para el diámetro promedio del anillo. Asuma normalidad en los datos

  • RESPUESTA
  • GRÁFICA
Ver código 
# Datos
x_barra <- 74.036
sigma <- 0.001
n <- 15
alpha <- 0.01

# Valor crítico
z <- qnorm(1-alpha/2)

# Error estándar
EE <- sigma/sqrt(n)

# Intervalo
LI <- x_barra - z*EE
LS <- x_barra + z*EE

# Respuesta
cat("IC =", LI, ",", LS)
IC = 74.03533 , 74.03667
Ver código 
confianza <- 0.99
alpha <- 1-confianza
z <- qnorm(1-alpha/2)

x <- seq(-4,4,length=1000)

df <- data.frame(x=x,y=dnorm(x))

sombra <- subset(df,x>=-z & x<=z)

ggplot(df,aes(x,y))+
geom_line(color="darkgreen",size=1.2)+
geom_area(data=sombra,fill="lightgreen",alpha=0.8)+
geom_vline(xintercept=c(-z,z),linetype="dashed")+
labs(title="IC 99% Distribución Normal Z")+
theme_minimal()

  • GRÁFICA

Suponga que las ventas de los dos grupos se distribuyen normalmente con varianzas de 48 para los titulados superiores y 56 para los de estudios secundarios.   • RESPUESTA
  • GRÁFICA
Ver código 
# Datos
x1 <- 17
x2 <- 19

s1 <- 1.5
s2 <- 1.8

n1 <- 14
n2 <- 25

alpha <- 0.01

dif <- x1 - x2

LI <- dif - t*EE
LS <- dif + t*EE

cat("IC =", LI, ",", LS)

alpha <- 1-confianza
gl <- 37

df <- data.frame(x=x,y=dt(x,df=gl))

sombra <- subset(df,x>=-t & x<=t)

geom_line(color="navy",size=1.2)+

22 .Ejercicio 9 - Varianza de la población

Una empresa desea analizar la variabilidad del tiempo de entrega de sus pedidos. Se seleccionó una muestra aleatoria de 18 entregas y se obtuvo una varianza muestral de (s^2=16) horas². Suponiendo que los tiempos se distribuyen normalmente, construya un intervalo de confianza del (99%) para la verdadera varianza poblacional.

  • RESPUESTA
  • GRÁFICA
Ver código 
# Datos
n <- 18
s2 <- 16

alpha <- 0.01

chi1 <- qchisq(alpha/2,df=n-1)
chi2 <- qchisq(1-alpha/2,df=n-1)

LI <- ((n-1)*s2)/chi2
LS <- ((n-1)*s2)/chi1

cat("IC =", LI, ",", LS)
IC = 7.615109 , 47.74261
Ver código 
confianza <- 0.99
alpha <- 1-confianza
gl <- 17

chi1 <- qchisq(alpha/2,gl)
